PHP & MySQL

The code in this folder is to accompany our PHP & MySQL book.

To run the code on your computer, you need to install a web server and database. The easiest way to do this is:

  • Windows: Download XAMPP then use these setup instructions
  • macOS: Download MAMP then use these setup instructions

For the code in Section C and D you need to have created a MySQL database using the SQL files in the code download, and you need to update the database connection details in the code for each chapter.
